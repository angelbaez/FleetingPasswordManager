Instructions for Fleeting Password Manager

General

Fleeting Password Manager is a program that generates pseudo-random passwords from given master password, URL/ID and user name. The master password should be common for all passwords and URL/ID should be the url of the service (e.g. www.facebook.com) or some part of it (e.g. facebook). Fleeting Password Manager then combines the master password, URL/ID and the given user name and generates the corresponding password. The user then changes her password to the one that Fleeting Password Manager generated. The user needs to remember only her master password. Other passwords are generated on-the-fly from the given login data when needed.

The generated passwords are always the same with the same input and are not stored to anywhere, which is actually the whole point of Fleeting Password Manager.

The length of the generated password is configurable between 8 and 32 characters and Fleeting Password Manager can remember wanted URL/User-pairs.

Features

  • Simple user interface
  • Combined password generator and manager
  • On-the-fly password generation corresponding to the login data (Passwords are never saved in any form)
  • Password length configurable from 8 to 32 characters
  • The generated password is being shown for a configurable period of time
  • Portable Qt-based source code

Examples

Example 1

For login information:

Master password:qwerty123
URL/ID:www.facebook.com
User name:john.doe@example.com

The generated 8-char password is always: OGM2ZjVk.

This password can then be copy-pasted to the facebook login field.

Example 2

Notice, that the URL doesn't have to be the real URL, it just has to be exactly the same every time, something that you remember. You can use "gmail" for Gmail, "facebook" for www.facebook.com, "linkedin" for www.linkedin.com and so on.

For login information:

Master password:qwerty123
URL:facebook
User:john.doe@example.com

The generated 8-char password is now always: ZjQxOWM1.

Exporting saved logins

As mentioned before, Fleeting Password Manager never saves any passwords. However, user name and url-pairs can be saved. On Linux, this data is located as a simple text file at:

~/.config/fleetingpm/fleetingpm.conf

If you copy this file to the same location in some other Linux-based system, you'll get the same saved logins.
